# Screenshots

Drop launch screenshots here, then wire them into the READMEs.

## Needed

- `screenshot.png` — the hero image used in `README.md` / `README.ko.md`
  (a board view with a few colorful cards reads best; ~820px wide).

## How to wire it in

In both `README.md` and `README.ko.md`, uncomment the image line under the header:

```html
<img src="docs/screenshot.png" alt="Kabin screenshot" width="820" />
```

## Optional extras

- `board.png`, `table.png`, `dashboard.png` — one per view, for a small gallery.
- `demo.gif` — a short drag-and-drop capture for the top of the README.
